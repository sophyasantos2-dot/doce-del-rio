O HTML (HyperText Markup Language) é o esqueleto do site. Imagine que vai construir uma casa: o HTML não é a cor da parede nem a fiação elétrica, são os tijolos
e as vigas que definem onde é o quarto e onde é a sala.Ele é uma linguagem de marcação. Ele não pensa nem faz cálculos (isso é com o JavaScript), ele apenas diz
ao navegador: "Isto aqui é um título", "Isto aqui é uma imagem".
As tags são os comandos que avisam ao navegador o que deve aparecer na tela.
<h1> a <h6>: São os títulos. O <h1> é o título principal (o patrão da página) e os outros vão diminuindo de tamanho e importância até o <h6>.
<p>: É a tag de parágrafo. Usas para qualquer texto normal e corrido.
<a>: Abreviação de anchor (âncora). É a tag que cria links. Sem ela, não conseguias clicar em nada para ir para outro site.
<img>: Serve para colocar imagens. Ela precisa de um "caminho" (URL) para saber qual foto mostrar.
<ul> e <li>: Usadas para criar listas (como aquelas de supermercado, com bolinhas).
A <div> é, basicamente, uma caixa invisível.Ela serve para agrupar elementos. Imagina que quer que um título, um texto e um botão fiquem todos juntos num canto 
da página. Tu coloca todos eles dentro de uma <div>.É o ato de colocar uma tag dentro da outra. A <div> facilita muito isto, permitindo que tu organizes o 
código em blocos lógicos (topo, meio e fundo do site).Sem as <div>s, o código ficava todo solto e era impossível organizar o layout do site de forma bonita 
depois.

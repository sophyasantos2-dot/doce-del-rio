# Anotações do Projeto

Esta pasta contém anotações, resumos e notas técnicas do projeto Doce del Rio.

## Arquivos

- **resumo.md** - Resumo geral do projeto e funcionalidades
- **resumo_css.md** - Anotações sobre estilos CSS utilizados

## Adicionar novas notas

Para adicionar novas anotações:

1. Crie um arquivo `.md` nesta pasta
2. Use nomes descritivos (ex: `validacoes.md`, `performance.md`)
3. Documente as decisões tomadas e os motivos

## Exemplo de estrutura

```markdown
# Título da Nota

## Contexto
Descrever o problema ou situação

## Solução Implementada
Descrever a solução

## Referências
Links úteis, documentação, exemplos
```

🏗️ ARQUITETURA IMPLEMENTADA
═══════════════════════════════════════════════════════════════════════════════

Camadas (Onion Architecture):

    ┌─────────────────────────────────────────────┐
    │         Presentation (View)                 │
    │  - form.php, sucesso.php, erro.php         │
    └──────────────────┬──────────────────────────┘
                       │
    ┌──────────────────▼──────────────────────────┐
    │    Application (Controller)                 │
    │  - Orquestra requisição → Service → View   │
    │  - try-catch profissional                  │
    └──────────────────┬──────────────────────────┘
                       │
    ┌──────────────────▼──────────────────────────┐
    │    Business Logic (Service)                 │
    │  - Valida regras de negócio                │
    │  - Lança BusinessRuleException             │
    └──────────────────┬──────────────────────────┘
                       │
    ┌──────────────────▼──────────────────────────┐
    │    Data Access (Repository)                │
    │  - Implementa Interface                    │
    │  - Contém todo SQL                         │
    │  - Hidratiza entidades                     │
    └──────────────────┬──────────────────────────┘
                       │
    ┌──────────────────▼──────────────────────────┐
    │    Infrastructure (Database)               │
    │  - Singleton PDO                           │
    │  - Config centralizado                     │
    └─────────────────────────────────────────────┘

🔐 SEGURANÇA IMPLEMENTADA
═══════════════════════════════════════════════════════════════════════════════

✅ Banco de Dados
   • Singleton: Apenas uma conexão PDO
   • Prepared Statements: Previne SQL Injection
   • Config isolado: .gitignore protege config.ini

✅ Validação de Entrada
   • filter_var(): Validação de tipo e range
   • filter_sanitize_string(): Remove tags HTML
   • filter_input(): Limpa dados POST
   • Whitelist de cursos permitidos

✅ Tratamento de Erros
   • BusinessRuleException: Erros esperados
   • Exception genérica: Erros inesperados
   • Sem Stack Trace ao usuário final
   • Debug mode configurável

✅ Arquitetura
   • Desacoplamento total entre camadas
   • Injeção de Dependência obrigatória
   • Interfaces para contratos
   • Responsabilidade única por classe
